1. ทำนาย (Predict)

โมเดลรับ Features (ขนาด, ห้องนอน, ทำเล) แล้วคำนวณราคาตาม algorithm ที่เลือก เช่น Linear Regression

2. เปรียบเทียบ (Loss)

นำผลการทำนายเทียบกับราคาจริง 6,000,000 ฿ คำนวณความคลาดเคลื่อน ยิ่ง Loss สูงยิ่งคลาดเคลื่อนมาก

3. ปรับพารามิเตอร์

ใช้ค่า Loss ปรับตัวเลขภายในโมเดล (Gradient Descent) ให้ทำนายแม่นขึ้นในรอบถัดไป
