1. ทำนาย (Predict)

โมเดลรับ Features (x₁ x₂ x₃) คงที่ แล้วคำนวณ ราคา = w₁×120 + w₂×3 + w₃ ด้วยค่า w ปัจจุบัน

2. เปรียบเทียบ (Loss)

นำผลทำนายเทียบราคาจริง 6,000,000 ฿ คำนวณความคลาดเคลื่อน ยิ่ง Loss สูงยิ่งคลาดเคลื่อนมาก

3. ปรับ w₁ w₂ w₃

ใช้ค่า Loss ปรับน้ำหนักโมเดล (Gradient Descent) → Features เดิม + w ใหม่ = ผลทำนายแม่นขึ้นรอบถัดไป
