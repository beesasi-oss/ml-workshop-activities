Scatter Plot: ขนาดบ้าน vs ราคาบ้าน — ลากเพื่อวางเส้น
คลิกและลากบน Scatter Plot เพื่อวางเส้น Regression ของคุณ

RMSE คืออะไร?

Root Mean Squared Error = รากที่สองของค่าเฉลี่ยของ (ราคาจริง − ราคาทำนาย)²
ยิ่ง RMSE ต่ำ โมเดลยิ่งแม่น

Best Fit Line คืออะไร?

เส้นที่ทำให้ผลรวมของ (ค่าจริง − ค่าทำนาย)² น้อยที่สุด sklearn ใช้วิธี Least Squares หาเส้นนี้โดยอัตโนมัติ

R² (R-Squared) คืออะไร?

สัดส่วนของความแปรปรวนที่โมเดลอธิบายได้
R² = 1.0 → โมเดลสมบูรณ์แบบ
R² = 0 → โมเดลไม่มีประโยชน์
